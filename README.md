# 🎮 Simon Game Challenge

A fun and interactive **Simon Memory Game** built using **HTML, CSS, JavaScript, and jQuery**.
The game tests the player's memory by generating color sequences that must be repeated correctly to progress through levels.

---

## 📌 Features

* 🎨 Interactive colorful UI
* 🔊 Sound effects for each button
* 🧠 Memory sequence challenge
* 📈 Increasing difficulty levels
* ⚡ Real-time feedback animations
* 📱 Responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* jQuery

---

## 🎯 How to Play

1. Press any key to start the game.
2. Watch the sequence shown by the game.
3. Repeat the sequence by clicking the colored buttons.
4. Each level adds a new color to the sequence.
5. The game ends if you click the wrong button.
